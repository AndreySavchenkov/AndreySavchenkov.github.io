Мой первый пост о GIT (23.07.2018)

Так как я только начал изучение технологии git и сайта github, то оставлю здесь небольшую подсказку по самым необходимым командам.

git add .
- добавить все файлы.
git commit -m""
- сделать коммит.
git push origin master
- запушить в текущую ветку.
git status
- проверка текущего состояния репозитория.
