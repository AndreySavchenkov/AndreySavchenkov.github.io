Мой первый пост о GIT (23.07.2018)

Так как я только начал изучение технологии git и сайта github, то оставлю здесь небольшую подсказку по самым необходимым командам.

git add .
- добавить все файлы.
git commit -m""
- сделать коммит.
git push origin master
- запушить в текущую ветку.
git status
- проверка текущего состояния репозитория.

Подсказка к созданию меню

  1. Размечаем html код. Меню - это список ссылок.
  2. Сбрасываем стили списка. Добавим класс "main-menu". Обнулим отступы "margin: 0; padding: 0". Уберем маркеры "list-style: none;"
  3. Делаем ссылки меню блочными ".main-menu a {display: block;}"
  4. При желании можно убрать подчеркивания ссылкам при помощи свойств "text-decoration: none;"
  5. Псевдокласс :hover (выбираем элемент, когда на него наведен курсор и кнопка мыши не нажата)
